1. Назначение ПО.
Программа служит для копирования выбранных файлов в выбранную папку
2.  Подготовка к работе
- Наличие файлов для копирования
- Наличие целевого каталога копирования
- Для запуска программы необходимо запустить файл hardload.exe
3. Работа с системой
Функции программы:
- Кнопка "найти и добавить файл в список" позволяет найти файл для копирования и добавить в список
- Кнопка "Удалить выделеные файлы из списка" позволяет удалить из списка для копирования выделеные курсором файлы
- Кнопка "выбрать папку, куда писать файлы" позволяет найти папку в которую будут скопированны файлы
- Кнопка "начать грузить" позволяет запустить процесс копирования выбранных в списке чекбоксом файлов в целевую папку
- Кнопка "остановить грузить" позволяет препвать процесс копирования
- Элемент "Строка справки" позволяет прочитать название выбраного указателем мыши элемента интерфейса
- Элемент "Список файлов" позволяет просмотреть выбранные файлы и выделить какие из них будут скопированны
- Элемент "Папка, куда писать файлы" позволяет просматривать название и адрес целево папки
